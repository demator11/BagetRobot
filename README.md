# Робот-картограф (Webots + Python)

## 🎯 Задача

Робот в среде с комнатами, коридорами и тупиками должен **за фиксированное время** покрыть датчиками максимальную площадь и построить карту (occupancy grid).  
Без участия оператора, только алгоритмы.

**Метрики успеха:**
- Процент покрытия арены за отведённое время
- Точность карты (совпадение с реальной планировкой)

## 🧠 Архитектура

| Слой | Задача                                 | Ожидаемый результат |
|------|----------------------------------------|----------------------|
| 1. Алгоритм исследования | Решать, куда ехать дальше              | Frontier-based / жадный / случайный выбор цели |
| 2. Построение карты | Превращать показания датчиков в карту  | Occupancy grid (вероятностная) |
| 3. Избежание тупиков | Не застревать, не кружить              | Выход из dead-end / anti-oscillation |
| 4. Оптимизация | За ограниченное время покрыть максимум | Минимум повторных проездов, smart path |

## 🛠 Технологии

- **Симулятор**: Webots R2023b / R2024
- **Язык**: Python (контроллеры Webots)
